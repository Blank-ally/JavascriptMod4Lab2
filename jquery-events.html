jQuery Events

Due to the fact that jQuery objects are not HTMLElement objects, they will have different members (properties and methods). This means that methods such as .addEventListener will not work on jQuery objects. We can still add events to elements selected via jQuery using the .on method.

.on() method

The on method is to a jQuery object as the addEventListener method is to the HTMLElement object. Much like the addEventListener method, on expects an argument for the type of event as well as an argument for the function that will be executed when the event is triggered. 

$('#myButton').on("click", function(e){
    console.log(this);    
});
