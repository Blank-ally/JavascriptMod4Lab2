The jQuery Function

Documentation

The jQuery function creates a new jQuery object.

jQuery function: 

jQuery()

jQuery function shorthand: 

$()

You can use any CSS selector as a parameter inside of the jQuery function and it will find the matching element and return it as a jQuery object.

Example

The following selects every element with the class of active and return a jQuery object for the selection: 

$('.active')

Look at the console window for the logged result.
