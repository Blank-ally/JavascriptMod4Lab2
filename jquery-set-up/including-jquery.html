Including jQuery

Documentation

jQuery is a JavaScript library. A library is a collection of code that has already been created to provide additional functionality. jQuery is not natively included in JavaScript, meaning that it needs to be added to our projects. We can download the library (.js files) or we can include a link to a Content Delivery Network that has the files on their server.

CDNs offer a way to serve content to a user without having to host the content on your site. CDNs are often a faster and more reliable way to load content making them a good alternative to hosting content on your site.

jQuery is hosted on several CDNs. Here are a couple:



Google CDN

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.3/jquery.min.js"></script>

jQuery CDN

<script src="https://code.jquery.com/jquery-3.6.3.min.js" 
    integrity="sha256-pvPw+upLPUjgMXY0G+8O0xUf+/Im1MZjXxxgOcBQBXU=" crossorigin="anonymous"></script>

Try adding the Google CDN link to this HTML page. You will need to do this on each page you want to use jQuery. Remember that scripts load from top to bottom meaning that you should load jQuery before you use it.
